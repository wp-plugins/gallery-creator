Gallery Creator is a plugin which reveals hidden gallery settings in WordPress.

== Description ==
Gallery Creator for instance makes it possible to set a one-column gallery with 
medium sized images. This makes it very easy and fast to insert a large number 
of big images than doing these one-by-one.

== Installation ==
This section describes how to install the plugin and get it working.

1. Upload the `gallery-creator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
